## Libraries documentation

* Assertion api: [Chai BDD api]

* Coverage: [Istanbul]
* Mock api: [Sinon.JS]
* BDD Mock api: [sinon-chai]

[JSHint]: http://jshint.com/docs
[Istanbul]: https://github.com/gotwarlost/istanbul
[Sinon.JS]: http://sinonjs.org/docs
